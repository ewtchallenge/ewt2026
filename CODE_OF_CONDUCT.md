- No plagiarism - all code must be original
- Give credit for external libraries and resources
- Help fellow participants when possible
- Constructive feedback only
- Participants can discuss approaches and challenges
- No direct code sharing until after submission deadline

## Reviewer Guidelines

- Be respectful of every participant
- Prioritise constructive feedback, no harmful or abusive language
- No discrimination
- Review should be as helpful as possible but it should also encourage exploration and research by the participant
- When leaving a comment about a topic, concept, approach, please include relevant links to the documentation
- Prioritise code not yet reviewed by a human reviewer. Our goal is to review as many submissions as possible.
- Feel free to add additional review or context to any AI review.
- Ask for clarification from participant by tagging them when possible
